# **App Name**: MoooApp

## Core Features:

- Date & Weather Display: Display the current date and local weather in the header.
- Daily Quote Generator & Display: Generate a random daily quote, display it on a beautifully animated card, and prevent repetition. Use swipe gestures for navigation and allow users to share the quote with a MoooApp watermark. The AI will function as a tool to filter quotes for repetition.
- Interactive Hub: Display an interactive hub with shortcuts (four cards) that dynamically link to different features in the app: 'Reminders', 'Emotions Visualizer', 'Sleep Tracker', 'Pain Diary', and 'Creative Space'.
- Emotions Visualizer: Display floating 3D spheres, where each represents an emotion (color-coded). Allow users to drag and toss the spheres with realistic physics and collision effects. Tapping a sphere should open a journal modal, and include a floating button to add new emotion bubbles.
- Organization Hub: Create an organization hub with four clickable cards: 'Today’s Reminders' (checklist with confetti), 'Day-off Suggestions' (self-care tasks), 'Today’s Plan' (goals), and 'Tomorrow’s Plan' (pre-planning).  Gentle sounds when push notifications trigger an alarm.
- Creative Space: Creative Space: create an area for Free journaling within a Text area and Drawing Canvas (finger-drawing on pastel-colored canvas), displaying random messages to boost mood.

## Style Guidelines:

- Background: subtle animated glow, black base with soft purple light particles.
- follow the attached collor palette
- Font: pure white, rounded sans-serif, clean and calming.
- Font: 'PT Sans', a humanist sans-serif providing a balance of modernity and warmth, ideal for both headlines and body text within the wellness context.
- Use soft, rounded icons to maintain the Claymorphism design and gentle feel of the app.
- Employ Claymorphism with soft, puffy UI elements that appear like clay, using large border radius (16-20px) and soft inner and outer shadows. Rounded cards and subtle micro-interactions (e.g., confetti bursts, bouncing bubbles).
- Implement subtle micro-interactions and animations such as confetti bursts, bouncing bubbles, and smooth transitions between screens to enhance the app’s playful and responsive feel.
